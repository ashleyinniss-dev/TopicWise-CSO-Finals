# TopicWise-CSO-Finals

TopicWise is a web application created to help Caribbean students prepare for their CSEC examinations in a more organized and less stressful way. It brings study planning, exam countdowns, and learning resources together in one place.

What It Does
	•	Lets students choose their exam session and subjects
	•	Helps build a weekly study planner using drag and drop
	•	Shows live countdowns to upcoming exams
	•	Provides access to textbooks, past papers, SBAs, and other study resources
	•	Allows students to download their study planner for offline use

Built With
	•	HTML
	•	CSS
	•	JavaScript

How to Use

Open home-tab.html in your browser and navigate through the site using the menu.

NOTE: The website is not completed (resources need to be added) and animations are not finished as yet, these features will be completed in time for the 14th January, 2025 for the finals of the CSO Coding Olympiad!
